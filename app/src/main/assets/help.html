Conn Rail Help

 

Summary

Conn Rail provides the ability to determine the delivery locations of freight cars without the need for index cards or printed switch lists.

 

Setup involves two steps:

  1. Create a list of all "spots" on the layout. These are industries, storage tracks, yards, etc. in a town - any location a car can be left.
  2. Create a list of all freight cars to be used on the layout - initials, number and type. For each car a "spot cycle" is set up. This is a list of the sequence of destinations for that car. At each spot the number of days to hold the car can be set. Typically this is how long it takes to load or unload the contents of the car. Optionally, the car's lading (empty, coal, grain, lumber, etc.) and any special instructions can be set up as well.

 

Prior to an operating session, each car's actual location is set using Car Location. The car's location could be at a spot in a town, in a train on the layout, or in storage.

 

Once the operating session begins the user can select either the Train Master or Yard Master run options.

 

Train Master allows an engineer/conductor to view all the cars in their train and determine where each car needs to be delivered. They can also find which cars are available for pickup and add them to their train.

 

Yard Master allows a local switcher or freight agent to view the location of all the cars in their town and their destinations. Using this information the yard switcher can make up blocks of cars for pickup and the local switcher can deliver cars to their destinations.

Spot List

The first step is to examine the layout and list every location where a car can be left. Includes the following:

  • Industries, freight houses, team tracks, etc.
  • Interchange tracks, freight yards or other on-line holding tracks.
  • Staging, fiddle or other off-line holding tracks.

 

If a stretch of the mainline is used to leave cars for another train to pick up then that would be listed too. However, if you just use that same stretch of mainline to temporarily hold cars while performing switching activities then it doesn't need to be listed.

Each spot requires a City/Town. For offline/staging locations this is usually sufficient. For online locations the industry can be added. This could be the industry name such as "U.S. Grain" or some other descriptor such as "Yard", "Team Track", "Interchange" or "Staging". If the industry has more than one track the track name or number can also be included. This could be "1", "2" or "Inner", "Middle", "Outer", etc. Here are examples from our sample layout:

 

 

 

 


Note that the Westport Junction (WJ) yard doesn't have each individual yard track listed. For this layout, "somewhere in the yard" is sufficient. Other layouts may require a car to be left on a specific yard track or may have a large yard where adding track information will make it easier to locate a car. In that case track information would be added.
